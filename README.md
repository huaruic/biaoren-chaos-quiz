# 镖人·乱世行

基于电影《镖人》的沉浸式性格测试 Web 应用，带你穿越隋末乱世，发现内心深处的侠客原型。

## 简介

这是一款以隋末乱世为背景的沉浸式性格测试游戏。用户以第一人称视角经历一段从「风起」到「立道」的完整乱世旅程，通过 6 道情境选择题，探索你内心的侠客人格：

- **知世郎** - 星火的引路人
- **刀马** - 孤勇的守护者
- **阿育娅** - 坚韧的远行者
- **谛听/老莫** - 暗影的洞察者
- **燕子娘** - 秩序的执行者
- **玉面鬼** - 乱世的谋局者

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Motion (Framer Motion)
- Lucide Icons

## 本地运行

**前置条件:** Node.js

1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动开发服务器：
   ```bash
   npm run dev
   ```

3. 打开浏览器访问 http://localhost:3000

## 构建

```bash
npm run build
```
